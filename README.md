smarty-debug-chrome (beta)
===================

A pleasant way to debug your smarty variables within Chrome Dev Tools

====

## Usage
1. Open debug.tpl in your Smarty folder and replace the content with /extra/debug.tpl
2. Open modifier.debug_print_var.php in your Smarty folder and replace the content with /extra/modifier.debug_print_var.php
3. Make sure that smarty_debug is enabled
4. Install the chrome extension (url not yet available)
5. Open Dev Tools and click on the 'Smarty' panel
